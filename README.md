# MongoAttack
> 基于python2.7.x 的mongoDB攻击工具

## Usage
### 启动:
```
配置文件修改：
mongoattack/Lib/setting.py

$ cd mongoattack/
$ python mongoattack.py
```
### 依赖:
```
colorama (0.3.7)
shodan (1.6.4) 
requests (2.13.0) 
pymongo (3.4.0) [本地要装mongoDB]
```
